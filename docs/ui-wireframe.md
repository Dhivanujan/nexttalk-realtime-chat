# UI Wireframe (Text)

Desktop
- Shell (100% height)
  - Left Sidebar (30%)
    - Brand + user avatar
    - Search input
    - Conversation list (avatar, name, preview, time, unread badge)
    - Contacts / directory toggle
  - Chat Panel (70%)
    - Header (avatar, name, status, call icons)
    - Pinned bar
    - Message stream
      - Incoming left bubble, outgoing right bubble
      - Timestamp + status ticks
    - Composer (emoji, attach, mic, input, send)

Mobile
- Sidebar view
- Chat view
- Swipe/Back to return to list
