# Component Breakdown

## Sidebar
- App logo + user summary
- Conversation search
- Conversation list
- Contact directory toggle

## ChatWindow
- Header: avatar, name, status, call actions
- Message stream
- Pinned messages rail
- Typing indicator

## MessageBubble
- Sender name (group/channel)
- Body + media
- Timestamp + delivery status
- Reactions

## InputBar
- Emoji toggle
- Attachment control
- Voice recording
- Message input
- Send action
